# 5 分钟看完一份报告

> 每天 GitHub Actions 跑完会在 `07-每日复盘归档/YYYY-MM-DD.md` 写一份新报告。这是怎么读它的导读。

## 阅读优先级（从上到下，时间紧只看前 2 段）

### ① 30 秒 · 🎯 今日最终建议（报告最顶部）

```
🟢 BUY: N  ·  🟡 WATCH: N  ·  🔴 AVOID: N
```

下面一张表，每行一只标的：

| 含义 | 看到这个怎么办 |
|---|---|
| 🟢 BUY | 当前价在入场区内，论点没被红队枪毙 → **可以执行**（按报告里的仓位/止损/期权结构） |
| 🟡 WATCH | 当前价偏离入场区 → **不动**，挂条件单或等回踩 |
| 🔴 AVOID | 已破止损 OR 红队 🚨 OR 拿不到价 → **绝对不要碰** |

如果整张表全是 🔴，或者上方有 `🚨 红队发现 CRITICAL 问题`，**今天什么都不要做**。

### ② 1 分钟 · 📊 实时快照 + 5 日趋势

每只标的一行：
- **当前价**：~15 分钟延迟（Yahoo Finance）
- **5 日趋势**：`▁▂▃▄▅▆▇█` Unicode sparkline + 趋势百分比
  - 例 `▁▃▆█▇ +5.2%` = 一路上涨然后小幅回落
  - 例 `█▇▅▃▁ -8.1%` = 单边下跌（**警惕**）
- **区间**：5 日 Low / High
- **成交量**：当日成交（M = 百万股）

用这段验证 🎯 顶部的 verdict 是否合理。

### ③ 1 分钟 · 📰 各标的最新新闻

每只票 3 条 Yahoo 头条。任何「downgrade」「miss」「probe」「lawsuit」要把对应标的的 BUY 自己降一级。

## 上面看完，知道当日要不要动手了。下面是想深读时再看：

### ④ 3 分钟 · 5 层漏斗（报告主体）

| 层 | 看点 | 跳过条件 |
|---|---|---|
| Layer 1 催化剂 | 今天市场最关心的 3-5 件事 + 来源 URL | 如果你昨天已经看过宏观新闻可跳 |
| Layer 2 供应链节点 | 把催化剂映射到产业链位置 | 想理解 logic 时看 |
| Layer 3 瓶颈强度评分 | 7 维度打分明细 | 想质疑某个推荐时看 |
| Layer 4 社交信号 | KOL 论点 + 散户情绪 | 看 Serenity 当前在喊啥 |
| Layer 5 动作清单 | 期权结构 / DTE / 仓位 | 真要下单时仔细看 |

### ⑤ 2 分钟 · 自审批注（红队复核）

报告底部。红队（更强的模型）专门挑漏洞：

- ✅ 没有 🚨 → 推荐相对干净
- 🚨 CRITICAL ISSUES FOUND → 顶部 verdict 已自动降级，但**自己也要读一遍**理由

## 决策清单（每天执行 5 步，5 分钟）

- [ ] 看 🎯 顶部 verdict 表
- [ ] 看 📊 5 日 sparkline 是否合理
- [ ] 扫 📰 新闻有没有红旗
- [ ] 看自审 🚨 有没有亮
- [ ] 对照自己持仓上限 / 单板块 35% → 决定执不执行 BUY 项

---

## 反指标（看到立即跳过当日）

- 🎯 顶部全是 🔴 或全是 🟡
- 自审 🚨 出现，且未给出明确反驳
- 多只标的 5 日 sparkline 形态相同（说明只是大盘 beta，不是个股 alpha）
- 财报盘后日：避开当日所有期权裸卖
