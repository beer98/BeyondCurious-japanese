# 00 · 框架与方法论

## 一、为什么模仿 Serenity（@aleabitoreddit）

公开信息中，他做对了三件事，并且**可被结构化复制**：

1. **逆向拆解供应链**：不看股价 K 线，而是从终端需求（Hyperscaler 资本开支、AI 模型训练 token 增长）一层层倒推到上游卡脖子的元器件 / 材料 / 设备厂；
2. **找"未被注意的瓶颈"**：当一个环节出现产能 < 需求的剪刀差，且尚未被大行 cover，他在那里建仓；典型案例如 $AXTI（高纯铟）、$LITE/$COHR（CPO 光模块）、$SLOIF（Soitec）；
3. **期权卖方择时**：在已建立基本面信仰的票上，用 sell put / sell call spread 在波动率高位收割时间价值；他自己的"Bottleneck ETF"一周 +12.83%，等权重，每只都涨。

> 关键引语（自他 X 帖）："I trade unknown bottlenecks." / "Reverse-engineer the supply chain, then size up before institutional rotation."

## 二、我们的 5 层筛选漏斗

每天从 ~5000 只美股压缩到 3–5 只可执行标的：

```
Layer 1：宏观催化剂扫描（~2 小时窗口）
   ↓ 把"今天市场最关心的事"映射到产业链节点
Layer 2：供应链节点定位
   ↓ 找该节点下产能受限 / 替代品稀缺的公司
Layer 3：瓶颈强度打分（0–10）
   ↓ 估值 + 技术面 + 期权链流动性
Layer 4：社交信号交叉验证（X / Reddit / StockTwits）
   ↓ Serenity / 头部 KOL 是否提过；情绪冷热（逆向指标）
Layer 5：风险预算 + 期权结构选择
   ↓ 出"今日动作"
```

### Layer 1 · 宏观催化剂扫描

**信息源（每日必扫）：**
- 财报日历（Earnings Whispers / Finviz）+ 当周 SEC 8-K
- 美联储 / 经济数据（CPI、NFP、FOMC）
- 行业大会：GTC、Hot Chips、CES、JPM Healthcare、OFC（光通讯）
- 关键人物推文：Jensen Huang、Sam Altman、Elon、Cathie Wood、Serenity 本人
- Hyperscaler 资本开支变化：MSFT / GOOG / META / AMZN / ORCL
- 地缘 / 关税 / 出口管制：BIS Entity List、China 商务部公告

**输出：今日 3–5 个最强催化剂的关键词。**

### Layer 2 · 供应链节点定位

把催化剂翻译成"哪一个产业环节会被影响"。常用映射表（持续扩展，存于 `08-知识沉淀/供应链地图.md`）：

| 催化剂关键词 | 受影响节点 | 候选标的 |
|---|---|---|
| AI 算力 / token 增长 | HBM、CoWoS、CPO、电源 | $MU $SK Hynix $TSM $LITE $COHR $VRT $PWR |
| 推理本地化 / Edge AI | RISC-V、NPU、低功耗内存 | $SIFY $ARM $MU |
| 中国稀土 / 关键金属管制 | 镓、锗、铟、稀土磁材 | $MP $TMC $AXTI $USAR |
| 电网瓶颈 | 变压器、HVDC、铜 | $ETN $HUBB $GEV $SCCO |
| 核能 / SMR | 浓缩铀、核燃料 | $CCJ $LEU $SMR $OKLO |
| GLP-1 / 减肥药放量 | 多肽 CDMO、注射笔 | $LLY $NVO $CTLT $WST |

### Layer 3 · 瓶颈强度评分（0–10）

每只候选打分，加权求和：

| 维度 | 权重 | 评分要点 |
|---|---|---|
| 产能 / 需求剪刀差 | 30% | 行业产能利用率 ≥85%，新增产能 lead time ≥12 月 |
| 替代难度 | 20% | 客户切换成本高、设计 lock-in |
| 估值安全垫 | 15% | 1× P/B 或 <2× P/S（深度折价加分） |
| 财务质量 | 10% | FCF margin、净负债率 |
| 技术形态 | 10% | 周线在 50W MA 上方 + 缩量回踩 |
| 期权链流动性 | 10% | OI 厚 + IV 偏度合理 |
| 机构关注度（**反向**） | 5% | 卖方 cover 越少越好 |

**总分 ≥7.0 进入 Layer 4。**

### Layer 4 · 社交信号交叉验证

- **X**：搜 `$TICKER from:aleabitoreddit OR from:PhotonCap OR from:zerohedge`，看最近 7 天有无新论点
- **Reddit**：r/wallstreetbets、r/stocks、r/options 的 daily thread
- **StockTwits**：情绪冷热（>80% bullish 时反向警惕，<30% bullish 反而是机会）
- **Google Trends**：相关供应链关键词搜索量是否抬头

**判定规则：**
- KOL 已喊单 + 散户冷淡 → 最佳（早期布局窗口）
- KOL 喊单 + 散户狂热 → 减半仓位（拥挤）
- 无 KOL + 散户冷淡 + 基本面强 → 加仓（最高价值）
- 无 KOL + 散户狂热 → 跳过

### Layer 5 · 风险预算 + 期权结构

**仓位规则：**
- 单一标的 ≤ 总仓位 15%
- 单板块（如 CPO）≤ 35%
- 总杠杆 ≤ 1.5×（含期权 delta 暴露）

**期权选择树：**
```
基本面强 + IV 高 + 看涨 → Sell Cash-Secured Put（30–45 DTE，Δ≈0.30）
基本面强 + IV 低 + 看涨 → Buy Long Call 或正股
基本面强 + 已持有 + IV 高 → Sell Covered Call（Δ≈0.20）
基本面弱 + IV 高 → Sell Call Spread
不确定方向 + IV 极高 → Iron Condor 或不动
```

## 三、自进化机制（核心）

每周日跑一次"经验蒸馏"：
1. 把 7 天内所有交易和未执行的候选，按"判断 → 结果"对齐；
2. 找出**系统性偏差**（例如：连续 3 周高估某板块、连续 3 次错过 Layer 4 的 Reddit 信号）；
3. 把校正写成新规则，**追加**到 `08-知识沉淀/规则库.md`；
4. 下周开盘前 Claude 自动加载规则库——这就是进化。

**反规则同样重要**：被打脸的判断要写"以后看到 X 不要 Y"，比正向规则更值钱。

## 四、心法（不要忽略）

- Serenity 反复说："淋浴时灵感（shower thoughts）当作严肃分析框架"——意思是**保持产业链直觉，不被日内噪音绑架**。
