# 01 · 每日工作流 SOP

> 时间均为 **美东时间 (ET)**。中国用户请 +13 / +12（夏令时）。

## 一、盘前研究（07:00 – 09:30 ET）

### 07:00–07:30 · 信息汇集（30 min）
- [ ] 查看隔夜亚洲 / 欧洲市场（Nikkei / DAX / Hang Seng）涨跌前 5
- [ ] 看 **Earnings Whispers**：今日盘前 / 盘后财报清单
- [ ] 扫 **Finviz News**：过去 12 小时全球宏观大事
- [ ] 看 **X 时间线**：@aleabitoreddit、@PhotonCap、@zerohedge、@WallStJesus
- [ ] 看 **r/wallstreetbets Daily Discussion** + r/stocks Daily
- [ ] 检查持仓：是否触发预设止损 / 止盈

### 07:30–08:30 · Layer 1–3 漏斗（60 min）
- [ ] 把今日催化剂写到 `02-每日研究模板.md` 顶部
- [ ] 用 Claude 跑 `05-Claude执行提示词.md`，输入催化剂关键词
- [ ] AI 输出 8–12 个候选 → 人工剔除明显噪音 → 留 5 个进 Layer 4

### 08:30–09:15 · Layer 4–5（45 min）
- [ ] 每只候选搜 X / Reddit 情绪
- [ ] 看期权链：当周 + 月内 IV、Skew、Open Interest 集中点
- [ ] 决定结构（正股 / call / put / spread）
- [ ] 写"今日动作清单"，每条带：标的、方向、入场区间、仓位、止损、止盈、退出条件

### 09:15–09:30 · 最终检查
- [ ] 与昨日"知识沉淀"对照，是否触犯反规则？
- [ ] 设置 IBKR / TOS 条件单
- [ ] 写"最大遗憾测试"：如果今天什么都不做会怎样？如果今天满仓会怎样？

## 二、盘中（09:30 – 16:00 ET）

**默认不交易**。只在以下情况动手：

- 触发盘前预设的条件单 → 让算法执行，不手动改
- 出现盘前未预料的"瓶颈级新闻"（典型：出口管制、并购、重大产能事故）→ 重跑 Layer 1–3，写一份**临时研究**追加到当日文件
- 持仓触止损 → 立即清仓，不商量

**禁止：**
- 盯盘超过 30 分钟连续（导致冲动交易）
- 在 X / Reddit 看到喊单立即跟车
- 在 IV crush 当日（财报后）裸卖期权

## 三、盘后复盘（16:00 – 17:30 ET）

### 16:00–16:30 · 数据收集
- [ ] 记录今日已执行交易的最终成交价、滑点
- [ ] 截图持仓 P&L 与 Greeks
- [ ] 看盘后财报反应（如有）

### 16:30–17:30 · 写当日复盘
打开 `07-每日复盘归档/YYYY-MM-DD.md`（用 `02-每日研究模板.md` 复制），填：

- ✅ **命中**：哪些判断对了？为什么？
- ❌ **未命中**：哪些错了？归因到框架的哪一层？
- 🤔 **观察**：今天看到但没建仓的信号，明天还要跟吗？
- 🆕 **新规则候选**：是否需要写入知识沉淀？

### 周五额外：周复盘
- [ ] 跑 `04-自进化复盘机制.md` 的周复盘流程
- [ ] 更新 `08-知识沉淀/规则库.md` 与 `供应链地图.md`

## 四、月度（每月最后一个交易日）

- [ ] 计算当月夏普 / 最大回撤 / 胜率 / 平均 R:R
- [ ] 与基准（SPY、QQQ、SOXX）对比
- [ ] 重审仓位上限（如连续两月跑输 → 缩仓 50% 直到回归）
- [ ] 给系统打分：1–10 分，写改进清单

## 五、不可妥协的纪律

1. **写下来才算数**——没写在当日文件里的判断不算预测，事后找补无效
2. **止损是物理操作不是心理操作**——挂 GTC 止损单，不靠意志力
3. **复盘必须当天做**——隔夜记忆会篡改归因
4. **AI 是顾问不是法官**——任何 AI 给的建议，自己想不通就不做
